# Enforce-Phase-I-Governance

## Description
This skill acts as a gatekeeper before and during implementation, preventing violations of specifications, constitution, or phase scope while ensuring quality standards are maintained.

## Parameters
- `spec`: Current feature specification
- `constitution`: Project constitution
- `phase_scope`: Phase I boundaries
- `quality_metrics`: Quality standards to enforce

## Execution
1. Validate current work against spec
2. Check constitution compliance
3. Verify phase scope boundaries
4. Assess quality metrics
5. Block or approve implementation

## Output
- Compliance status report
- Violation identification
- Implementation approval or block
- Quality assessment
