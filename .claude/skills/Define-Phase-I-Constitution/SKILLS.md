# Define-Phase-I-Constitution

## Description
This skill creates or updates the project constitution for Phase I, establishing non-negotiable principles, constraints, and boundaries for the project.

## Parameters
- `principles`: Core principles that guide the project
- `constraints`: Technical and business constraints
- `boundaries`: Clear scope boundaries for Phase I

## Execution
1. Read existing constitution if present
2. Apply new principles while maintaining consistency
3. Validate against project goals
4. Update `.specify/memory/constitution.md`

## Output
- Updated constitution file
- Confirmation of applied principles
