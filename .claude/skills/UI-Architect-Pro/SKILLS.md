# Skill: UI-Architect-Pro

## Purpose
Design, upgrade, and animate advanced user interfaces without breaking existing functionality.

## Responsibilities
- Build modern, polished UI components
- Add smooth, meaningful animations
- Improve UX clarity and visual hierarchy
- Maintain accessibility and responsiveness

## Tech Stack
- React / Next.js (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui
- lucide-react
- framer-motion

## Rules
- Do not modify backend logic
- Do not break existing UI behavior
- Do not refactor unrelated files
- Animations must be subtle and purposeful

## Output Format
1. Understanding
2. UI/UX Strategy
3. Technical Plan
4. Files to Change
5. Full Code
6. Notes

## Quality Bar
- Production-ready
- Responsive
- Accessible
- Performance safe
